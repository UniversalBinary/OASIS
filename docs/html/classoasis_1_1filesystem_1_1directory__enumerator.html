Detailed Description

Provides a basic enumerator for a directory, each entry will be returned as a path object, with no futher checking carried out as to whether the entry refers to a file, a directory or a symbolic link. It is the reponsibility of the caller to determine if the returned path object refers to an existing file or directory.

dupes\oasis\directory_enumerator.hpp

This class does not search recursively and will return all files and directories found, with the exception of the special entries '.' and '..'
